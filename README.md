# Money-VS-Happiness-
Money vs Happiness — World Happiness Analysis (2015–2024)


"Can money actually make you happy? I analyzed 10 years of global data to find out."


📌 Overview
Analyzed 150+ countries across 2015–2024 using the World Happiness Report dataset to explore what truly drives happiness — wealth, freedom, or something else entirely.

🔍 Key Findings

💛 GDP (0.78) — strongest happiness driver globally
🟢 Freedom (0.55) — matters more than generosity
🔴 Corruption (-0.43) — trust in govt destroys joy
🏆 Finland — happiest country, multiple years running
🇮🇳 India — peaked in 2017, dropped post-COVID
